Como usar esta interface

  • Dashboard — visão geral: último ciclo, métricas e tarefa atual.
  • Ciclos — histórico completo de ciclos com resultados expandíveis.
  • Mensagens — chat com o agente. Mensagens são injetadas no próximo ciclo.
  • Carteira — saldo e divisão de lucros 80/20.
  • Recibos — auditoria de ferramentas executadas.

Solicitar mudanças de código

Para solicitar alterações no código do agente (main.py, tools, etc.), descreva a mudança no campo de mensagens com prioridade urgent. As mudanças de código são realizadas por Claude Code na sessão ativa — cole o pedido no chat do Claude Code.
